## Annotation Workflow

### 1. Image Assessment
- Review image metadata and available context  
- Assess image quality (positioning, exposure, artifacts)  
- Identify key thoracic anatomical landmarks  

### 2. Pathology Identification
- Perform systematic chest X-ray review  
- Apply clinical reasoning to identify abnormal findings  
- Use standardized radiology terminology  

### 3. Annotation Application
- Apply appropriate annotation type (bounding boxes, segmentation masks, classification)
- Assign severity and structured attributes where applicable 
- Maintain consistent labeling conventions  
- Ensure anatomical and pathological accuracy  

### 4. Quality Assurance
- Self-review for completeness and precision  
- Spot-check annotations for consistency  
- Document ambiguous findings and edge cases  

### 5. Documentation
- Log annotations and decisions in tracking spreadsheets  
- Record QA observations and corrections  
- Maintain reproducible annotation guidelines

## Limitations & Scope

This portfolio project was designed to demonstrate clinical annotation skills and workflow discipline, rather than to represent a large-scale production dataset.

Scope
- Focused annotation of 50 chest X-ray images
- Coverage of common thoracic pathologies
- Explicit severity grading using defined radiographic thresholds
- Inclusion of structured clinical attributes such as air bronchograms, meniscus sign, vascular markings, and tracheal deviation
- Annotations performed by a single annotator (self)  
- Intended primarily for portfolio and skill demonstration purposes  

Limitations
- No formal inter-annotator agreement (IAA) metrics calculated  
- Annotation counts and statistics are approximate  
- Dataset size is not sufficient for standalone model training  
- Image interpretation performed without access to full clinical reports  

Despite these limitations, the project reflects real-world annotation practices and demonstrates readiness to contribute effectively to larger, collaborative healthcare AI annotation projects.
